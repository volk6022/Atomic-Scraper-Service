# Доводка схемы/промптов по W2 (итог)

Правки в `yandex_enrichment_experiment/02_research_orgs.py` (референс для порта в
auto-monitor). Валидация — на эндпоинте (фикснутый агент), сравнение с backup.

## Что сделано
1. **`build_org_card_schema(categories)`** — `tech_stack` включается в схему только
   для IT/digital-категорий; для остальных убирается. Закрывает schema_gap (141 орг
   в анализе 517, где tech_stack массово пуст/нерелевантен).
2. **`build_query`** — соцсети вынесены в приоритет №1 с явной целью «найти DM-канал
   (профиль + @-хэндл)» и обязательным скрейпом найденных соцсетей; tech-строка —
   только для tech-категорий.
3. Схема прокинута по категории (`run_research`/`process_org`/`verify_run.py`).

## Валидация
**Схема-фикс: чистый детерминированный выигрыш** — `tech_stack` отсутствует во всех
картах не-tech орг (Хлебник, Аптека, Кикимора, Подземка). Без даунсайда.

**Соц-акцент — эффект в пределах шума (n=4):**
| орг (тип) | social backup → v3 | DM-хэндлы |
|---|---|---|
| Хлебник (сеть) | 3 → 6 ✓ | да |
| Подземка (одиночная) | 5 → 6 ✓ | да (@BarPodzemka…) |
| Кикимора (одиночная) | 6 → 3 ✗ | да |
| Аптека 36,6 (сеть) | 2 → 0 ✗ | нет |

2↑/2↓ по числу — локальная 9B сильно варьирует между прогонами. Качественно v3
стабильнее достаёт **DM-хэндлы** (@-ники, t.me-ссылки) — то, что нужно для аутрича.
Строгий A/B потребовал бы 20–30 орг × (старый vs новый query) ≈ часы GPU.

## Главный смежный вывод (дешевле, чем доводить промпт)
Соцсети — это канал аутрича, а самый **надёжный и дешёвый** их источник —
**карточка организации Яндекс.Карт**: в SSR-объекте есть поле `socialLinks`
(VK/WhatsApp/…), достаётся одним httpx-GET ~100 КБ **детерминированно**
(см. `parse-yandex-economy-experiment` эксп. 9–11). То есть для соц-контактов
не нужно полагаться на стохастический LLM-ресёрч — можно тянуть из Я.Карточки
напрямую, а LLM использовать для остального (описание, вакансии, масштаб, проблемы).

## Рекомендация
- Схема-фикс — принять (готов к порту).
- Соц-query — оставить (направленно полезен, достаёт хэндлы; вреда нет), но в
  проде дополнить **детерминированным извлечением `socialLinks` из Я.Карточки** —
  это закроет «канал аутрича» надёжнее любого промпта.
- Замечена высокая дисперсия одиночных прогонов локальной 9B → для важных орг
  имеет смысл 2 прогона с мержем или поднять порог критика.

Артефакты v-прогонов: `verify_endpoint/results/` (v3), `results_v2/` (фикс-агент,
старый query), `backup_cards/`.
