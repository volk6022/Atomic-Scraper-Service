# Infrastructure Layer

Модуль для внешних взаимодействий.

## Структура
- `browser/` - Менеджеры браузеров (Pool и Session)
- `llm/` - LLM клиенты (OpenAI, Jina)
- `queue/` - Taskiq брокер и воркеры

## Ответственность
- Управление жизненным циклом браузеров
- Интеграция с внешними API (LLM)
- Очереди задач и воркеры
