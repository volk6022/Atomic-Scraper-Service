# Browser Module

Управление браузерами Playwright.

## Структура
- `pool_manager.py` - Синглтон для Stateless задач (живёт весь цикл воркера)
- `session_manager.py` - Индивидуальный браузер для Stateful сессий

## BrowserPoolManager
- Запускается один раз при старте воркера
- Создаёт изолированные контексты для каждой задачи
- Закрывается только при остановке воркера

## SessionBrowserManager
- Создаётся для каждой Stateful сессии отдельно
- Управляет жизненным циклом сессии
- Закрывается по таймауту или принудительно
